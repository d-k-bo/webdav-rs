# webdav-headers

<!-- cargo-rdme start -->

HTTP headers for WebDAV as defined in
[RFC 4918](http://webdav.org/specs/rfc4918.html#http.headers.for.distributed.authoring)
implementing the [`headers::Header`] trait.
